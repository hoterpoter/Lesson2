## Списки

Чтобы добавить ненумерованый список,
необходимо выделить пункты (*)

Например:
* Элемент
* Элемент
* Элемент

Чтобы добавить пронумерованый список,
необходимо пронумеровать списки.

Например:

1. Элемент
2. Элемент
3. Элемент
