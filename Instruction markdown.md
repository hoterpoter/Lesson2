# Инструкция для работы с Markdown.

## Выделение текста

Чтобы выделить текст курсивом необходимо обрамить его (*)

пример: *курсив*

Чтобы обрамить текст жирным шрифтом необходимо обрамить его (**)

пример: **полужирный**
